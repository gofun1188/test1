物資捐贈說明

物資捐贈說明
Readme

【舊鞋救命捐贈物資方法】
​
【方法一】 親自送至伯利恆倉庫
地址：新北市林口區中山路578號（伯利恆倉庫）
親送時段：星期一〜星期六 10：00~18：00，週日及國定假日休倉。

【方法二】 指定募集期間寄送
為了讓倉庫志工與夥伴能在可預期時程內整理完成，請勿於非募集期寄送包裹，謝謝您的體諒。
​寄送地址：新北市林口區中山路578號（伯利恆倉庫）
掌握每次募集時間，請加入 @LINE
或撥打電話： 03-426-1783
可使用郵局、嘉里大榮物流、新竹物流、黑貓宅急便，以及各大貨運物流皆可寄送。

若您需要捐物收據，您可以直接點選 申請捐物收據 連結後填寫相關資料，我們會於當月月中開立上個月捐款/捐物收據(有接收到收據開立申請者)，並整理寄出。(例：8月份的捐款/捐物收據，9月中旬開立整理寄出)

​【倉庫大眾交通方式】

  1、從台北前往倉庫的民眾可以到捷運圓山站(圓山轉運站)，搭程客運937或是前往捷運府中站搭客運948，兩條路線皆可在文化三路中山路口下車，再步行約 5 分鐘即可到達林口倉庫。

2、其它縣市民眾可以先搭機捷到捷運林口站再轉搭936或854路線公車至文化二路中山路口下車後，步行約 7 分鐘後即可抵達林口倉庫。


 

​【捐贈注意事項】

【童鞋】：十二歲以下童鞋最迫切需要。
【舊鞋】：功能完整之男鞋﹑女鞋( 平底包鞋為主 )，鞋底保有彈性能九十度彎曲、無破洞、裂痕、脫皮、氧化等狀況。
【舊衣】：以輕薄衣物為主（ 衣服褲子大人小孩皆可；內衣褲，襪子不收 ）。
【書包】：畢業書包、帆布提袋以及其他可放入 A4 尺寸書本之各式包款（ 環保袋、不織布、硬殼包、拉桿包不收 ）。


【所需物資】

​舊鞋：
​完整包覆性鞋款，不限尺寸、年齡、性別之布鞋、球鞋、皮鞋、娃娃鞋、粗低跟鞋、短靴、軍靴。
功能正常，無破洞、磨損、裂痕、脫皮、氧化等狀況。
1.請於清潔後用鞋帶對綁，沒有鞋帶可用塑膠繩、棉繩綑綁，兩兩成對不分離。請勿使用橡皮筋，易斷裂反而失散。
2.物資盡可能裸裝，切勿使用塑膠袋與鞋盒包裝，一個紙箱內可放多雙鞋子，不要過多的包裝，例如：不需要一雙鞋一個塑膠袋、一雙鞋一個鞋盒。
3.孩童的學步鞋需堅硬的底，如布面軟底不適合。

舊包：
畢業書包、後背包、側背包、帆布袋、束口背袋、以及其他可放入 A4 尺寸書本之各式包款或提袋，若無法放入 A4 尺寸書本約兩三本，則太小的不適合。
*環保袋、不織布、硬殼包、拉桿包不收。

舊衣：
​1.以輕薄衣物、褲子、裙裝，大人、小孩皆可。
2.無標註個人資料之學校、公司制服可捐。
3.衣物請清潔、檢查、整理過後再捐出。
4.內衣褲、襪子不收。（ 全新也不收 ）
5.大片泛黃、污漬、散發成年氣味，請使用過碳酸鈉浸泡熱水去除，仍無法去除請勿捐贈。
6.破損衣物，請勿捐贈。
＊因文化的不同，刷破設計的牛仔褲，不適合當地請勿捐贈，過於曝露或過短的裙子、褲子，如迷你裙、熱褲，亦不適合。


【非所需物資】 

鞋子：
​不耐穿的款式、拖鞋、涼鞋、高跟鞋、厚底鞋、內增高鞋、長靴、雪靴、雨鞋、布希鞋、塑膠鞋、洞洞鞋。
鞋子失去彈性、脫膠、破洞、鞋底斷裂、龜裂、破碎、發霉、掉底、開口、鞋面氧化......等。
*以上款式因氣候、環境與耐用性等因素不適合，請勿捐贈。

包包：
不能裝A4尺寸之包包、零錢包、塑膠材質、硬殼材質、不織布購物袋、紙袋、塑膠袋、商品袋、拉桿式書包、行李箱。
*以上款式皆因環境、氣候、實用性、以及貨櫃運送堅硬物資，容易擠壓破損碎裂，請勿捐贈。

衣物：
​過度暴露、過短,如：熱褲、迷你裙。貼身衣物、內衣褲、襪子、泳衣、主要功能為保暖的冬衣、帽子、圍巾、皮帶配件。
*以上款式皆因環境、氣候、文化因素不適合，請勿捐贈。

 

    

   

      

【物資捐贈聲明】

舊鞋救命國際基督關懷協會，所有募集的物資將先經過志工整理打包，除去不適用非洲之愛心捐物轉贈至台灣境內慈善組織外，所有資源將優先運送至發展中國家，並逐步以社會企業方式，改善基本生活現況，以基督的愛關懷非洲赤貧地區，並以宣教、教育、飲水、衛教等方式，逐步建立和諧分享的自立社區為目的。

舊鞋救命國際基督關懷協會保有權力決定物資發送時間地點，並尊重當地使用物資的意願與方式。或在特殊狀況下將物資轉換為醫療設備、運費、勞務或家畜等其他資源，進一步執行人道關懷工作，以改善赤貧地區居民基本生活為目的。
